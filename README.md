# 류성국 시의원 후보 홈페이지

더불어민주당 창원시 다선거구(명곡·봉림) 류성국 후보 소개용 원페이지 홈페이지입니다.

## 작업 기준

- Next.js + TypeScript + Tailwind CSS
- GitHub Pages 정적 배포
- 후보 정보와 공약은 확인된 자료만 표시
- 확인 전 연락처, 선거사무소, 공약 세부안은 준비 중 처리

## 실행

```bash
npm install
npm run dev
```
